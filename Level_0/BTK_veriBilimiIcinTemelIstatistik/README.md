🇬🇧This repository contains Python implementations of statistical concepts covered in the "Veri Bilimi için Temel İstatistik" (Fundamental Statistics for Data Science) course offered by BTK Akademi.
The goal of this project is to reinforce theoretical statistical concepts through practical coding exercises and explore fundamental statistical techniques commonly used in data science.
Topics Covered
Descriptive statistics
Probability theory
Bayes theorem
Statistical distributions
Central Limit Theorem
Hypothesis testing (Z-test, T-test, ANOVA)
Correlation analysis
Linear regression
Logistic regression
Principal Component Analysis (PCA)
Basic classification and clustering concepts
All examples are implemented in Python using widely used data science libraries such as NumPy, Pandas, Matplotlib, and Scikit-learn.
This repository serves as a personal learning archive and practical reference for statistical foundations in data science and machine learning.
